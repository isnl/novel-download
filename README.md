# 网文电子书下载系统

## 简介

本系统是一个网文电子书下载系统，支持爱下电子书、52书库等书源的下载，目前支持txt、epub文件格式。

## 技术栈

express + ts + unocss + ejs

## 书源

- 爱下电子书 ixdzs8.com
- 52书库  ltxs5.net
- 持续更新中......

## 安装使用

nodejs 20+  

```bash
git clone git@github.com:isnl/novel-download.git
```

```bash
npm install
```

```bash
npm run dev
```

open your browser and visit http://localhost:5030
